Project Overview

During the internship, the project I made was intended for use by architecture security analysts.

By combining architecture design with AI analysis, the efficiency and accuracy of employees in auditing projects can be increased.

The operating process of the software is as follows: users register and log in to their accounts.

They can select the corresponding project model to enter the project according to project requirements, build an architecture diagram as required, and select the version number of each component and the user's natural language description of the component.

After the architecture is built, AI analysis can be performed with one click.

AI will generate different threat analyses unique to each component based on the version selection and natural language description of each component, so as to help security analysis architects find vulnerabilities and threats of each component faster.
